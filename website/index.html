The launcher

Your whole modded setup, managed like light switches.

Point Omega at an existing Minecraft install — or an empty folder — and it takes care of the rest: real installs straight from Mojang, mods as toggles, and PvP-tuned launches.

Mods become switches, not folder surgery.

Import your own .jars once and flip them per-instance. Omega reads each jar's real metadata to show proper names, versions and descriptions — and auto-tags everything so presets can bulk-switch your setup for the game mode you're queueing.

  • Non-destructive toggling — disabling renames mod.jar to mod.jar.disabled; nothing ever leaves your folder
  • One-click presets — Smooth PvP, Crystal PvP, UHC, Bedwars, Survival, Visual/HUD only
  • Config editor built in — edit any mod's JSON or TOML config from a generated form, no text editor needed

A real install engine — nothing re-hosted.

The New Instance dialog downloads any Minecraft release straight from Mojang's own CDNs, with one-click Fabric or Forge on top. Every file is sha1-verified, downloads resume, and each launch is preceded by a genuine install-completeness check.

  • Bring your own install — auto-detects versions and loaders in any standard .minecraft layout
  • Real launch engine — resolves version JSON inheritance, builds the classpath, and spawns the JVM directly
  • Smooth-PvP JVM tuning — optional G1GC flag preset that cuts GC-pause frame hitches in close fights

Shaders and sign-in, handled for you.

A compatible shader loader — Iris + Sodium on Fabric, Oculus on Forge — is preinstalled automatically, so imported shader packs just work. Microsoft sign-in works out of the box with tokens encrypted at rest and refreshed before every launch.

  • Shaders tab — import .zip packs and pick the active one in-game
  • Multi-account — link several Microsoft accounts and switch per-instance
  • Auto-updates — installed builds update themselves in the background
